Conversion des fichiers Excel vers le format texte Amplitude

Instructions

Structure du fichier Excel

  • Ligne 1: En-têtes du donneur d'ordre
  • Ligne 2: Données du donneur d'ordre
  • Ligne 3: En-têtes des bénéficiaires
  • Lignes 4+: Données des bénéficiaires

Format généré

Le fichier texte généré respectera les spécifications Amplitude avec:

  • Un enregistrement "Entête Donneur d'ordre"
  • Des enregistrements "Détail virement bénéficiaire"
  • Chaque ligne de 255 caractères

Procédure

  1. Téléchargez votre fichier Excel
  2. Vérifiez les données extraites
  3. Saisissez un préfixe (optionnel)
  4. Générez et téléchargez le fichier texte
