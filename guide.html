🚀 Guide d'utilisation rapide

Générateur MT103 RTGS BCEAO - CORIS Bank UEMOA

Étape 1 : Sélection de la filiale

Choisissez la filiale CORIS qui recevra les fichiers MT103 RTGS dans la liste déroulante.

8 filiales disponibles : Burkina Faso, Bénin, Côte d'Ivoire, Guinée Bissau, Mali, Niger, Sénégal, Togo

Étape 2 : Configuration de base

  • Date de valeur : Sélectionnez la date des transactions
  • Nombre de fichiers : 1 à 100 fichiers
  • Transactions par fichier : 1 à 1000 messages MT103 par fichier

Étape 3 : Import des IBAN bénéficiaires

Option A - Import fichier :

  • Glissez-déposez votre fichier CSV/TXT
  • Ou cliquez sur "Parcourir"
  • Format : un IBAN par ligne

Option B - Saisie manuelle :

  • Sélectionnez "Saisie manuelle"
  • Entrez les IBAN, un par ligne
💡 Astuce : Un fichier exemple exemple_iban.txt est fourni avec 20 IBAN de test des 8 pays UEMOA.

Étape 4 : Configuration des montants

Définissez la plage de montants en XOF :

  • Montant minimum : Ex: 10 000 XOF
  • Montant maximum : Ex: 5 000 000 XOF

Préréglages rapides :

  • Petits montants : 5 000 - 100 000 XOF
  • Montants moyens : 50 000 - 1 000 000 XOF
  • Gros montants : 500 000 - 10 000 000 XOF

Étape 5 : Options avancées (optionnel)

Cliquez sur "Options Avancées" pour personnaliser :

  • Client donneur d'ordre : Nom du client émetteur
  • Information de paiement : Motif de paiement
  • Préfixe du nom de fichier : Personnaliser le nom des fichiers

Si laissé vide, ces informations sont générées aléatoirement de façon réaliste.

Étape 6 : Génération

Cliquez sur le bouton "Générer les fichiers MT103"

L'application va :

  • ✅ Valider toutes les données
  • ✅ Générer les messages MT103 conformes SWIFT/BCEAO
  • ✅ Afficher un aperçu des fichiers générés

Étape 7 : Téléchargement

Deux options de téléchargement :

  • Téléchargement individuel : Bouton sur chaque carte de fichier
  • Téléchargement groupé : Bouton "Télécharger tous les fichiers"

📋 Format du fichier IBAN

Exemple de fichier iban.txt ou iban.csv :

ML1810100200041802510100
BF1234567890123456789012
CI9876543210987654321098
SN1111222233334444555566
NE1234567890123456789012
TG1234567890123456789012
BJ1234567890123456789012
GW1234567890123456789012
💡 Bon à savoir :
  • Les IBAN sont automatiquement validés (format UEMOA)
  • Les lignes vides et caractères spéciaux sont ignorés
  • Si vous avez moins d'IBAN que de transactions, ils sont réutilisés en rotation

🎯 Cas d'usage typiques

Test de virements entrants

Configuration recommandée :

  • 1 fichier avec 10-20 transactions
  • Montants moyens (50 000 - 1 000 000 XOF)
  • Liste d'IBAN de comptes de test

Test de charge

Configuration recommandée :

  • 10-50 fichiers avec 100-500 transactions chacun
  • Large plage de montants
  • Grande liste d'IBAN variés

Formation équipe

Configuration recommandée :

  • 3-5 fichiers avec 5-10 transactions
  • Montants simples et arrondis
  • Options avancées personnalisées
⚠️ Important :
  • Ces fichiers sont générés pour tests uniquement
  • Ne jamais utiliser en environnement de production
  • Toujours valider dans votre CBS Amplitude de test d'abord
  • Les checksums et MAC sont générés aléatoirement

🔧 Dépannage

Erreur : "Aucun IBAN valide trouvé"

Solution :

  • Vérifiez que les IBAN commencent par un code pays UEMOA (BF, BJ, CI, GW, ML, NE, SN, TG)
  • Assurez-vous qu'il y a un IBAN par ligne
  • Supprimez les en-têtes de colonnes si présents

Erreur : "Le montant minimum doit être inférieur au maximum"

Solution :

  • Inversez les valeurs min/max
  • Utilisez les préréglages rapides

Les fichiers ne se téléchargent pas

Solution :

  • Vérifiez que votre navigateur autorise les téléchargements
  • Essayez le téléchargement individuel
  • Vérifiez l'espace disque disponible
